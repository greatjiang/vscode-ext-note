# 发布
1. npm i vsce -g 安装
2. vsce create-publisher your-publisher-name 创建发布者
3. vsce publish  发布
4. vsce unpublish  取消 
5. 如果关联github 需要先提交到github上
6. 手动npm version x.x.x
